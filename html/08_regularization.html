Write our own polynomial regression¶

Pay attention to the use of scipy.linalg.inv and np.float128. This is to ensure the correct inversion of the least-squares matrix.

In [4]:
def polynomial_regression(x, y, k):
    # use scipy inv insead of np inv, slight better performace
    from scipy.linalg import inv

    X = np.vander(x, k+1, increasing=True)

    # more float precision to deal with tiny coefficients in matrix inversion
    X = X.astype(np.float128)
    Y = y.astype(np.float128)

    res = np.dot(X.T, X)
    res = np.dot(inv(res), X.T)
    res = np.dot(res, Y)

    coef = res.squeeze() 
    
    return coef

Regularization¶

Regularization helps us keep (regularize) the "sanity" of the model by penalize the large coefficients.

There are two common types of regularization used in machine learning: L2 and L1. They are call Ridge regression and Lasso regression, respectively.

L1 norm: Manhattan Distance¶

L2 norm: Euclidean Distance¶

Before L2 and L1 regularization¶

Recall the least-sqaures minimization problem:

$$ \mathbf{X} \mathbf{\beta} = \mathbf{y} $$

The least square method will minimize the residual sum of squares (RSS) in matrix format as:

$$ RSS(\beta) = (\mathbf{y} - \mathbf{X} \beta)^T (\mathbf{y} - \mathbf{X} \beta) $$

In regularization, instead of minimize the RSS directly, we minimize the (RSS + Penalty Term).

The cost function becomes:

$$ Q(\beta) = RSS(\beta) + \text{penalty} $$

L2 regularization - Ridge regression¶

In Ridge regression, the penalty term is defined as:

$$ \alpha ||\beta||_2^2 $$

which is the square of L2 norm of all the coefficient. Here $\alpha$ is a scalar value and $\beta$ is a vector.

Then, we are going to minimize the new cost function, denoted as $Q(\beta)$:

$$ Q(\beta) = RSS(\beta) + \alpha ||\beta||_2^2 $$

This equals to:

$$ Q(\beta) = (\mathbf{y} - \mathbf{X} \beta)^T (\mathbf{y} - \mathbf{X} \beta) + \alpha \beta^T \beta $$

In order to obtain $\beta$, we need to set the gradient of $Q(\beta)$ to zero. The gradient of $Q$ can be calculated as:

$$ \nabla Q(\beta) = \nabla (\mathbf{y} - \mathbf{X} \beta)^T (\mathbf{y} - \mathbf{X} \beta) + \nabla \alpha \beta^T \beta $$

In earlier section, we already know the gradient of the first terms, and the gradient of the second term is simply $2 \alpha \beta$, we have:

$$ \nabla Q(\beta) = -2 \mathbf{X}^T(\mathbf{y} - \mathbf{X} \beta) + 2 \alpha \beta $$

Set it to zero, we can calculate the regularized coefficients as:

$$ -2 \mathbf{X}^T(\mathbf{y} - \mathbf{X} \beta) + 2 \alpha \beta = 0 \\ \beta = (\mathbf{X}^T \mathbf{X} + \alpha I)^{-1} \mathbf{X}^T \mathbf{y} $$

where $I$ is the identity matrix.

Python implementation¶

Now we can construct our own Ridge regression function:

def polynomial_regression_l2(x, y, k, alpha):
    from scipy.linalg import inv

    X = np.vander(x, k+1, increasing=True)

    X = X.astype(np.float128)
    Y = y.astype(np.float128)
    
    res = np.dot(X.T, X)
    res = res + alpha * np.identity(k+1)
    res = np.dot(inv(res), X.T)
    res = np.dot(res, y)

    coef = res.squeeze()
    
    return coef

from sklearn.pipeline import make_pipeline

L1 regularization - Lasso regression¶

In Lasso regression, instead of L2 norm penalty, we use the L1 norm penalty

$$ \alpha ||\beta|| $$

Then, we are minimizing the new cost function $Q(\beta)$:

$$ Q(\beta) = RSS(\beta) + \alpha ||\beta|| $$

Which becomes:

$$ Q(\beta) = (\mathbf{y} - \mathbf{X} \beta)^T (\mathbf{y} - \mathbf{X} \beta) + \alpha ||\beta|| $$

In order to obtain $\beta$, we need to set the gradient of $Q(\beta)$ to zero. The gradient can be calculated as:

$$ \nabla Q(\beta) = \nabla (\mathbf{y} - \mathbf{X} \beta)^T (\mathbf{y} - \mathbf{X} \beta) + \alpha ~ sign(\beta) = 0 $$

Where $sign(\beta)$ is the element-wise sign of $\beta$.

Because of this, there is no "clean" closed form algebraic solutions, as compared to L2 regularization.

Commonly, we can use the gradient descent (coordinate descent) to obtain the solution.

We are going to leverage the Scikit-Learn package to perform L1 regression.
